# Dual Layer
This is a dual layer page. It has two layers of content that can be toggled between. The first layer is a chart, and the second layer is the code. A toggle button is provided to switch between the two layers. The chart layer is displayed by default, and the code layer can be accessed by clicking the toggle button. The code layer contains the source code for the chart, which can be copied and pasted for use in other projects.
